# Web-Application-for-eSIM-Distribution-
A web based application for accessing eSIM QR codes

═══════════════════════════════════════════════════════════════════════════
eSIM DISPENSER V2 - Production Version
═══════════════════════════════════════════════════════════════════════════
  
DESCRIPTION:
Internal use only-script for distributing esims across remote team.
ICCID numbers and QR code URLs manually entered into a hidden inventory sheet.
 
FEATURES:
- Email validation (@telus domain required)
- Account number validation (numeric only)
- Quantity limits (1-10 eSIMs per request)
- Automatic inventory tracking 
- Event logging
- Prevents SIM reuse through timestamp tracking
